# Genshinfy

Genshinfy is a non-commercial music streaming project based around music from Genshin Impact. Currently hosted on [Vercel](https://genshinfy.vercel.app/).

## Tech Stack
* Nuxt 3/Vue JS 
* Tailwindcss
* Typescript
* Firebase
* Howler JS
* Vee-Validate
